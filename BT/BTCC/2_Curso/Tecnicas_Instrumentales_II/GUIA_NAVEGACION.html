Flujos de Navegación Recomendados

📚 Para Estudiantes (Flujo de Estudio)

  1. Inicio (index.html) → Obtener visión general del curso
  2. Material Teórico (material_del_estudiante..html) → Estudiar Ficha 1 y 2
  3. Ejercicios (Ejercicios_Practica..html) → Practicar casos graduados (Ej. 1 al 4)
  4. Examen (Examen_Parcial..html) → Resolver el examen completo
  5. Autoevaluación → Verificar respuestas (si el docente proporciona solucionario)

👨‍🏫 Para Docentes (Flujo de Enseñanza)

  1. Inicio (index.html) → Acceso centralizado a todos los recursos
  2. Plan Anual (plan_de_clase..html) → Revisar calendario y capacidades
  3. Material Teórico (material_del_estudiante..html) → Preparar clases
  4. Ejercicios (Ejercicios_Practica..html) → Proponer prácticas en clase
  5. Examen (Examen_Parcial..html) → Aplicar evaluación
  6. Solucionario (Solucionario_Examen..html) → Calificar con rúbrica

Instrucciones de Uso

Abrir Archivos Localmente

  1. Asegúrese de que todos los archivos HTML estén en la misma carpeta.
  2. Haga doble clic en index.html para abrir en el navegador.
  3. Navegue usando los enlaces y menús.

Publicar en Línea (Servidor Web)

  1. Suba todos los archivos HTML a su servidor web.
  2. Mantenga la estructura de carpetas.
  3. Acceda a través de la URL (ej: www.tudominio.com/tecnicas_instrumentales_ii/)

Personalización

  • Para editar contenido: Modifique los archivos .md y reconvierta a HTML si es necesario.
  • Para cambiar colores: Busque los códigos de color (#0f4c81, #667eea) en los estilos CSS.
  • Para agregar páginas: Use el mismo template HTML y agregue enlaces en el index.html.
