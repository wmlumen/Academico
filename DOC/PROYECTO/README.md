# MEC PRO 2026: Ecosistema Pedagógico Digital

¡Bienvenido al sistema de gestión estratégica para docentes del Ministerio de Educación y Ciencias (Paraguay)!

## 🚀 Inicio Rápido
1.  Abre el archivo **[Dashboard_Docente.html](file:///c:/Users/HP%20250%20G10/Documents/GITHUT/MEC/Dashboard_Docente.html)** en tu navegador.
2.  Configura tus rubros y horarios.
3.  ¡Administra tu aula en tiempo real!

## 📂 Organización Logística Final (v19)
El repositorio ha sido optimizado con una estructura **App + Recursos**:

- **/Dashboard_Docente.html**: Portal de entrada único.
- **/Recursos/**: Carpeta maestra organizada por niveles:
    - `EEB/`: Educación Escolar Básica (7º, 8º, 9º).
    - `EM/`: Educación Media.
    - `BT/`: Bachilleratos Técnicos (Construcciones, Ambiental, Informática).
    - `ADM/`: Gestión Administrativa y Marco Legal.
    - `DOC/`: Especificaciones Técnicas y Roadmap del Proyecto.

## 🧹 Limpieza del Repositorio
Para purgar las carpetas duplicadas de la raíz (`7_Grado`, `8_Grado`, `Materiales_Organizados`, etc.) y dejar solo lo esencial:
- Ejecuta el script de PowerShell: **`.\Limpiar_Repositorio.ps1`**.
