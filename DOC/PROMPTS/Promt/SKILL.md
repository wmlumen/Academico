# Skill: Materiales pedagógicos reproducibles estilo MEC Paraguay

## Propósito
Usar este skill cuando el docente solicite crear, completar, adaptar o mejorar materiales pedagógicos para Educación Media, Bachillerato Técnico, EEB u otra modalidad, con lenguaje institucional, estructura clara y documentos listos para imprimir, copiar o editar.

Este skill NO copia literalmente documentos oficiales protegidos. Produce materiales propios, alineados a las capacidades, contenidos, indicadores y formatos que el docente proporcione o que se infieran de documentos curriculares subidos por el usuario.

## Activar este skill cuando el usuario pida
- Plan Anual de Cátedra.
- Plan de Clase, plan diario, secuencia didáctica o microplanificación.
- Instrumentos de evaluación: lista de cotejo, rúbrica, prueba escrita, guía de observación, escala estimativa, registro anecdótico, portafolio, RSA.
- Materiales para estudiantes: guía de aprendizaje, ficha de trabajo, lectura breve, ejercicios, práctica de laboratorio/taller, tarea domiciliaria.
- Planilla de proceso: seguimiento por estudiante, indicadores, puntajes, evidencias, observaciones, asistencia y recuperación.
- Paquete completo por materia, curso, unidad o mes.

## Datos mínimos que deben solicitarse si faltan
Antes de generar el paquete, identificar o pedir solamente los datos esenciales:
1. Institución.
2. Docente.
3. Área, disciplina o módulo.
4. Curso, sección, turno y año lectivo.
5. Especialidad, si corresponde.
6. Carga horaria semanal o cantidad de horas de la unidad.
7. Capacidades oficiales o competencias a trabajar.
8. Unidad, contenido o tema.
9. Periodo: mes, etapa, semana o fecha.
10. Tipo de documento requerido: Plan Anual, Plan de Clase, evaluación, material del estudiante, planilla de proceso o paquete completo.

Si el usuario no proporciona todos los datos, generar una versión editable con campos “___________________” y notas “Completar por el docente”.

## Principios de producción
- Mantener lenguaje formal, pedagógico y técnico.
- Organizar todo en tablas cuando sea útil para impresión.
- Usar verbos observables: identifica, describe, analiza, calcula, aplica, compara, resuelve, diseña, argumenta, ejecuta, registra, verifica.
- Alinear cada indicador con una capacidad.
- Alinear cada evaluación con al menos un indicador.
- Incluir evidencias de aprendizaje concretas: ejercicios resueltos, informe, maqueta, plano, práctica, exposición, prueba, portafolio, lista de cotejo.
- Incluir adecuaciones curriculares razonables cuando corresponda.
- Incluir seguridad, uso correcto de herramientas, ética, ambiente y trabajo colaborativo en materias técnicas.
- Evitar contenidos genéricos sin relación con la capacidad.

## Terminología preferida
Usar términos frecuentes en planificación escolar paraguaya:
- Capacidades.
- Contenidos.
- Estrategias didácticas.
- Indicadores.

- Evidencias.
- Instrumentos de evaluación.
- Registro Secuencial de Aprendizajes / RSA.

- Recuperación pedagógica.

## Salidas esperadas del paquete completo
Cuando el usuario pida “todo”, “completo”, “para reproducir”, “materiales MEC” o similar, generar como mínimo:

1. **Plan Anual de Cátedra**
   - Datos identificatorios.
   - Fundamentación.
   - Competencia general.
   - Capacidades.
   - Matriz anual: mes/etapa, capacidad, contenido, estrategias didácticas, indicadores, evaluación, recursos, tiempo.
   - Bibliografía.
   - Observaciones y firmas.

2. **Plan de Clase**
   - Datos identificatorios.
   - Tema.
   - Capacidad.
   - Indicadores.
   - Inicio, desarrollo y cierre.
   - Actividades del docente y estudiante.
   - Recursos.
   - Evaluación.
   - Tarea o actividad de extensión.
   - Adecuaciones.

3. **Instrumentos de evaluación**
   - Lista de cotejo.
   - Rúbrica analítica.
   - Prueba escrita o práctica.
   - Guía de observación.
   - Clave de corrección o criterios.

4. **Material para estudiantes**
   - Título y datos.
   - Propósito de aprendizaje.
   - Lectura o explicación breve.
   - Actividades guiadas.
   - Ejercicios de aplicación.
   - Actividad práctica o proyecto.
   - Autoevaluación.
   - Tarea.

5. **Planilla de proceso**
   - Nómina de estudiantes.
   - Indicadores evaluados.
   - Evidencias.
   - Puntaje o valoración.
   - Observación del docente.
   - Recuperación o refuerzo.

## Formato base del Plan Anual
Usar la siguiente matriz, adaptando la cantidad de filas al año lectivo:

| Mes/Etapa | Capacidades | Contenidos | Estrategias didácticas | Indicadores | Estrategias evaluativas | Recursos didácticos | Tiempo |
|---|---|---|---|---|---|---|---|
| Marzo | [Capacidad observable] | [Tema/unidad] | [Actividades del docente y estudiante] | [Criterios observables] | [Instrumento/evidencia] | [Materiales] | [Horas] |

Reglas:
- Cada fila debe ser coherente entre capacidad, contenido, actividad, indicador y evaluación.
- No repetir el mismo indicador en todo el año sin ajustar el nivel de complejidad.
- Distribuir contenidos de menor a mayor dificultad.

## Formato base para rúbrica analítica

| Criterio | Excelente 5 | Muy bueno 4 | Bueno 3 | A mejorar 2-1 | Puntaje |
|---|---|---|---|---|---|
| Dominio conceptual | Explica con precisión y usa vocabulario técnico. | Explica con leves imprecisiones. | Reconoce ideas principales. | Presenta confusiones importantes. |  |
| Procedimiento | Aplica todos los pasos correctamente. | Aplica la mayoría de los pasos. | Requiere apoyo parcial. | No logra completar el procedimiento. |  |
| Producto/evidencia | Completo, ordenado y verificable. | Completo con detalles menores. | Parcialmente completo. | Incompleto o sin evidencia clara. |  |
| Actitud y seguridad | Trabaja con responsabilidad y normas de seguridad. | Cumple casi siempre. | Cumple con recordatorios. | No cumple normas básicas. |  |

## Formato base para prueba escrita o práctica

**Datos**
- Institución: ___________________
- Disciplina: ___________________
- Curso: ___________________
- Estudiante: ___________________
- Fecha: ___________________
- Puntaje total: ____ puntos

**Indicaciones**
Lee atentamente cada consigna. Resuelve con orden. Justifica los procedimientos cuando corresponda.

**Ítems sugeridos**
1. Selección múltiple: conceptos básicos.
2. Completar: vocabulario técnico.
3. Verdadero/Falso con justificación.
4. Resolución de problema o caso práctico.
5. Producción breve: explicación, esquema, cálculo, procedimiento o informe.

**Clave de corrección**
Incluir respuesta esperada, puntaje por ítem y criterio de asignación de puntaje parcial.

## 1. Activación de saberes previos
Responder:
1. ¿Qué sabes sobre ___?
2. ¿Dónde se aplica este tema en la vida cotidiana o profesional?

## 2. Desarrollo conceptual breve
Redactar una explicación clara, con ejemplos simples, definiciones y vocabulario técnico.

## 3. Actividad guiada
Resolver paso a paso una situación inicial.

## 4. Actividad individual o grupal
Incluir ejercicios graduados: básico, intermedio y aplicado.

## Generación por materia técnica
Para materias técnicas como Construcciones Civiles, Materiales, Taller, Tecnología, Laboratorio, Topografía o Resistencia de Materiales:
- Incluir normas de seguridad.
- Incluir herramientas, equipos, materiales y EPP.
- Incluir actividad práctica o simulada.
- Incluir evidencia: croquis, cálculo, informe técnico, fotografía, planilla, maqueta, ensayo, plano o práctica de taller.
- Incluir control de calidad del producto.

## Comando maestro para generar paquete completo
Cuando el usuario escriba algo como “generar materiales para [materia]”, responder creando el paquete con esta estructura:

1. Portada del paquete.
2. Plan Anual.
3. Plan de Clase de la primera unidad.
4. Material para estudiantes de la primera unidad.
5. Instrumentos de evaluación.
6. Planilla de proceso.
7. Recomendaciones de impresión y reproducción.

## Checklist de calidad antes de entregar
- [ ] Tiene datos identificatorios.
- [ ] Las capacidades son observables.
- [ ] Los indicadores miden la capacidad.
- [ ] Las estrategias didácticas son realizables en aula/taller/laboratorio.
- [ ] La evaluación tiene instrumentos claros.
- [ ] Hay material para estudiante listo para fotocopiar.
- [ ] La planilla de proceso permite seguimiento.
- [ ] Incluye recursos, tiempo y adecuaciones.
- [ ] Usa lenguaje formal y editable.
- [ ] No afirma que el material sea oficial si fue generado por IA.
